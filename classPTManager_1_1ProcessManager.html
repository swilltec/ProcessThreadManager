Destructor that ensures all managed processes are cleaned up.

Terminates all running processes gracefully (SIGTERM), then forcefully (SIGKILL) if necessary, and waits for them to complete. Prevents zombie processes.

◆ terminateAll()

void PTManager::ProcessManager::terminateAll ( )

Terminates all managed processes gracefully, then forcefully if needed.

First attempts graceful termination by sending SIGTERM to all processes, waits 1 second for them to clean up, then sends SIGKILL to any survivors. Finally waits for all processes to be reaped to prevent zombies.

This two-phase approach allows processes to perform cleanup operations before being forcefully killed.
