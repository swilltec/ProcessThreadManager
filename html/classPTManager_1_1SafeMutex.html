bool PTManager::SafeMutex::lock ( std::chrono::milliseconds  timeout = std::chrono::milliseconds(5000))

Attempts to acquire the mutex with timeout and recursive lock detection.

Parameters
timeoutMaximum time to wait for the lock
Returns
true if lock was acquired, false on timeout or recursive lock attempt

Detects and prevents recursive locking by the same thread, which would cause deadlock. Uses timed locking to detect potential deadlocks when timeout expires. Updates owner thread ID and increments lock count on successful acquisition.
