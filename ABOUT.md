# About ProcessThreadManager

## Project Information
**ProcessThreadManager** is a modern C++20 library for managing processes and threads on FreeBSD systems.

- **Repository**: [github.com/swilltec/ProcessThreadManager](https://github.com/swilltec/ProcessThreadManager)
- **Documentation**: [swilltec.github.io/ProcessThreadManager](https://swilltec.github.io/ProcessThreadManager/)
- **License**: MIT


## Features
- Process lifecycle management
- Thread pool with work stealing
- Resource monitoring and tracking
- Type-safe C++20 interface
- FreeBSD optimized

## Technology Stack
- **Language**: C++20
- **Platform**: FreeBSD
- **Build System**: Make
- **Documentation**: Doxygen
- **Testing**: Custom test framework
