PTManager::RWLock::RWLock ( )

Constructs a reader-writer lock with writer preference.

Initializes counters for tracking active readers, writers, and waiting writers. Writer preference prevents reader starvation of writers.

◆ readLock()

void PTManager::RWLock::readLock ( )

Acquires a read lock (shared access)

Blocks if any writer is active or waiting. Multiple readers can hold the lock simultaneously. Writer preference policy: waits for all waiting writers to get their turn before allowing new readers.

◆ writeLock()

void PTManager::RWLock::writeLock ( )

Acquires a write lock (exclusive access)

Blocks until all readers and writers have released their locks. Only one writer can hold the lock at a time, and no readers are allowed. Increments waitingWriters to prevent new readers from acquiring the lock.

void PTManager::RWLock::writeUnlock ( )

Releases a write lock.

Decrements writer count. Preferentially wakes one waiting writer if any exist, otherwise wakes all waiting readers to allow concurrent read access.
