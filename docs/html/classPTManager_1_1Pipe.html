PTManager::Pipe::Pipe ( )

Constructs an anonymous pipe for inter-process communication.

Creates a unidirectional communication channel using the pipe() system call. The pipe consists of two file descriptors: fds[0] for reading and fds[1] for writing. Logs error to stderr if pipe creation fails.
