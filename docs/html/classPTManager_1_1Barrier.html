void PTManager::Barrier::wait ( )

Blocks until all threads reach the barrier.

When the last thread arrives (count reaches threshold), all waiting threads are released simultaneously. The barrier automatically resets for reuse. Uses generation number to handle spurious wakeups and barrier reuse.
